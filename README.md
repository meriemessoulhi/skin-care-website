# Skin Care Website

## Description

This project is a simple skincare website that provides tips and product suggestions based on different skin types.

## Features

* Show skincare advice (dry, oily, combination, sensitive)
* Filter products
* Contact form with validation
* Responsive design (mobile friendly)

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Use

1. Open the website
2. Choose your skin type
3. View tips and products
4. Use the contact form if needed
